# Usmrtitve v ZDA

Ta repozitorij vsebuje projekt, v katerem analiziram **usmrtitve v ZDA**. 
Projekt je pripravljen pri predmetu **Računalniška ogrodja v matematiki 2020/2021**
mojega študija v 1.letniku na Fakulteti za matematiko in fiziko, Univerza v Ljubljani.

Pri projektu uporabljam računalniški program _Mathematica._

**Načrt projekta**

1. _Pridobivanje podatkov_ - Uporabila bom Wolfram data repository.
2. _Analiza usmrtitev v ZDA po državah ter drugih dejavnikih_ - narisala bom nekaj grafov števila
usmrtitev glede na države, spol, starost ter raso.
